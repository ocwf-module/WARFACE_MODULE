# Патч-ноут WF MOD SDK 1.4.5

## Изменения в редакторе WF MOD SDK 1.4.5 версии 1.0.1, в сравнении с той версией, в какой она дошла до автора

1. В папку \[Game\] добавлены стандартные pak-архивы из CryEngine 3.3.8, включая папку \[Localized\];
2. PAK-архив \[Scripts.pak\] был распакован в директорию \[Game\];
3. Обновлён pak-архив \[Engine/Engine.pak\];
4. \[CryLevelConvert/CryLevelConvert.exe\] заменён на последнюю версию;
5. Добавлена программа \[CryLevelConvert/Mission_Extract.exe\];
6. Добавлена программа \[Tools/CryArchiver/7za.exe\];
7. Скрипт сущности \[ShootingTarget\] обновлён до версии клиента 1.2.14.33043;
8. Удалены необязательные файлы, оставлен только сам редактор;
9. Обновлён стандартный TOD для лучшей видимости при создании уровня;

## Изменения в редакторе WF MOD SDK 1.4.5 версии 1.0.2

1. Обновлён файл \[MSDK_Editor\system.cfg\] - в него были прописаны все квары, которые отличались от игровых, для более корректной графики и технических значений;
2. Обновлено большинство скриптов и сущностей, для более точного использования, правильного экспорта и программировании в FlowGraph, удалены те, которые не используются в оригинальном клиенте.
3. В директорию \[MSDK_Editor\Tools\\] добавлены инструменты для просмотра и редактирования контента:
* DDS_Unpacking - для добавления текстур из новых клиентов;
* FSB_Extractor - для просмотра FSB-файлов;
* LUA_Decompile - для расшифровки компилированных скриптов LUA;
* PakDecrypt - для расшифровки PAK-архивов разных версий Warface;

## Изменения в редакторе WF MOD SDK 1.4.5 версии 1.0.3

1. Обновлено приложение \[MSDK_Editor\CryLevelConvert\Mission_Extract.exe\]:
* добавлено условие наличия архива уровня \[terraintexture.pak\], для корректного цвета текстуры террейна на уровне;
* добавлены необходимые обработки ошибок;
* путь к файлам \[LightSettings.lgt\] и \[TimeOfDay.tod\] заменён на \[Setting\TOD_and_Lighting\\], чтобы они находились в одной папке;
